# Changelog

## 1.0.0 — 2026-06-29

- Initial release of UX Interaction Skill.
- Four-dimension analysis workflow: Useful / Usable / Equitable / Enjoyable.
- 12 named design patterns from leading fintech and trading apps.
- 10 anti-patterns with priority grading (P0 / P1 / P2).
- 5-category multi-path checklist for comprehensive design coverage.
- App reference playbook indexed by dimension and scenario.
- Bilingual output (Chinese + English) summary standard.
- `agents/openai.yaml` for Codex / OpenAI Agent Skills integration.
- `scripts/ux_scorecard.py` for local scoring with JSON input.
- `scripts/validate_skill.py` for skill structure validation.
- `references/ux-dialogue-protocol.md` for conversation and learning-mode behavior.
- Three example outputs: prediction market onboarding, invite/VIP page, demo conversation.
- `evals/test-cases.md` for trigger and behavior tests.
