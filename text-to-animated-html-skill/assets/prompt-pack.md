# Prompt Pack

Ready-to-copy prompts for common use cases.

## Full pipeline: topic → HTML

```
帮我把这个话题做成动画 HTML 页面：[话题/产品/事件]
请先帮我生成口播脚本，再识别关键词动画，最后生成小红书风格的 HTML。
```

```
用 text-to-animated-html，把以下内容做成适合在手机上看的 HTML：
[粘贴内容]
推荐一个合适的平台风格。
```

## Script only

```
帮我写一段 1分30秒 的口播脚本，话题是：[话题]
按照 0:00-0:08 钩子、0:08-0:22 介绍、0:22-1:08 主体、1:08-1:22 反转、1:22-1:30 留钩 的结构来写。
```

## HTML from existing script

```
我已经有口播稿了，帮我直接做成 [小红书/TikTok/X/Dark Pro] 风格的 HTML：
[粘贴脚本]
```

## Style-specific requests

```
做成小红书风格 HTML，暖白背景，内容是：[粘贴内容]
```

```
做成 TikTok 竖版爆款风格，每页只放一个核心词，内容是：[粘贴内容]
```

```
做成 Dark Pro 风格，适合财经数据展示，内容是：[粘贴内容]
```

## Post-preview adjustments

```
翻页速度再慢一点，渐入进入。
```

```
数字滚动可以慢一点，大概 3 秒。
```

```
第 [N] 页的 [某元素] 改成 [描述]。
```

```
改成深色版，其他内容不变。
```

```
把 [某页] 的 [某个动画] 改成 [想要的效果]。
```
